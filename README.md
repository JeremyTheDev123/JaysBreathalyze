# JaysBreathalyze
BAC command for users to set bac and run bac. Made for FXServers.

Features:

* Set bac. /setbac
* Search bac. /searchbac


Installation:

* Place JaysBreathalyzer in resource folder.
* Start JaysBreathalyzer in your server.cfg.
